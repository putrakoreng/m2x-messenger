M2X Messenger

M2X Messenger is an Android application based on OpenYMSG which implements a Yahoo! Instant Messenger Client.

It is free (and open source), you can download it as an APK.

An overview of the project

M2X Messenger is based on a modified version of OpenYMSG. It implements a Yahoo! Instant Messenger Client for Android. The motivation behind its development was the lack of open source Yahoo! clients for Android and my attempt to learn Android software development.

The project is currently on its Beta release and is stable enough for daily use. Many features have been added since the early Alpha releases and you can now enjoy a feature rich open source Yahoo! messenger. Also bear in mind that the minimum SDK required to run it is v8 (Android 2.1 and above). The latest version features a completely new UI design which should be compatible with various screen densities.

Features

At its Beta stage, I've tried to provide a somewhat stable and feature-rich messenger ready for daily use.Below is a list of these functionalities:
  • Swipable chat window
  • Stealth support
  • Chat history support
  • Complete friends list management
  • Yahoo! avatar support
  • Yahoo! smiley support
  • Customizable alerts and notifications
  • Simple friends list event logger
